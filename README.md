# Public Holiday Checker

A Python application to check public holidays for different countries. This tool allows you to check if a given date is a public holiday, list all holidays for a specific year, and get upcoming holidays.

## Features

- Check if a specific date is a public holiday
- List all public holidays for a given year
- View upcoming public holidays
- Support for multiple countries (USA, UK, Canada, Australia, India)
- Command-line interface and GUI options
- Caching for improved performance
